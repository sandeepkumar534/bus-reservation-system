# Bus Reservation System

A Java-based Bus Reservation System developed using Object-Oriented Programming concepts.

## Features
- Passenger Management
- Seat Reservation
- Ticket Booking
- AC Ticket Booking
- Non-AC Ticket Booking
- Booking Timeout
- Exception Handling
- Multithreading

## Technologies Used
- Java
- ArrayList
- OOP Concepts
- Exception Handling
- Multithreading
